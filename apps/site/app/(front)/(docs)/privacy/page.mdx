# Soukromí a nakládání s daty

Data o používání aplikace budou po anonymizaci použita pro její zkvalitnění, výzkumné účely a vývoj nových služeb.

## Odstranění dat

Uživatelská data můžete kdykoliv smazat prostřednictvím aplikace. O odstranění uživatelských dat také můžete zažádat prostřednitvím e-mailu na [cernockyd@phil.muni.cz](mailto:cernockyd@phil.muni.cz).

## Jaká data ukládáme

V naší databázi uchováváme data údaje vložené uživatelem. Jedná se zejména o:

- uživatelské jméno
- jméno a příjmení
- e-mailovou adresu
- příspěvky

Při využívání aplikace dále mohou být automaticky schromažďovány informace:

- návštěvnost stránek
- uživatelské události a akce na stránkách
