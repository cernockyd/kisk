# Soukromí a nakládání s daty

Data o používání aplikace včetně obsahu konverzací budou po anonymizaci použita pro její zkvalitnění, výzkumné účely a vývoj nových služeb pro knihovny.

## Odstranění dat

Uživatelská data můžete kdykoliv smazat prostřednictvím aplikace. O odstranění uživatelských dat také můžete zažádat prostřednitvím e-mailu na [knihovnaai@phil.muni.cz](mailto:knihovnaai@phil.muni.cz).

## Jaká data ukládáme

V naší databázi uchováváme data údaje vložené uživatelem. Jedná se zejména o:

- uživatelské jméno
- jméno a příjmení
- e-mailovou adresu
- konverzace a jejich obsah
- poskytnutá zpětná vazba

Při využívání aplikace dále mohou být automaticky schromažďovány informace:

- návštěvnost stránek
- uživatelské události a akce na stránkách

## Kde jsou data uložena

Při ukládání a zpracovávání dat využíváme následující služby:

- Databáze (provozovaná na vědecké infrastruktuře) - pro ukládání a zpracování veškerých zpracovávaných dat
- OpenAI API - pro zpracování a generování konverzací
